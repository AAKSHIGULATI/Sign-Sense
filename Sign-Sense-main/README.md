# Sign Sense ✋🤟  
A machine learning–powered application that translates sign language into text and audio, enabling more inclusive communication for the hearing and speech impaired.  

---

## 🚀 Features
- Real-time *sign language detection* and translation into text & audio.  
- Custom-trained *image classification model* using *Google Teachable Machine*.  
- Achieved *98% accuracy* on test gestures.  
- Built with *Python* and *OpenCV* for live camera feed processing.  
- Audio output for recognized gestures using *text-to-speech (TTS)*.  

---

## 🛠 Tech Stack
- *Languages & Libraries*: Python, OpenCV, NumPy  
- *Machine Learning*: Google Teachable Machine, TensorFlow/Keras  
- *Tools*: Image Processing, TTS Engine (pyttsx3 / gTTS)
